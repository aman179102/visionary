Welcome to Visionary, a powerful real-time image and video analysis application built with Next.js and TensorFlow.js. Visionary empowers users to perform various computer vision tasks directly in their browser, leveraging pre-trained machine learning models for seamless and interactive experiences.

## Core Features

- **Webcam Feed Integration**: Access and display a live video feed from the user's webcam for real-time analysis.
- **Image Upload**: Allow users to upload local image files for classification and detection.
- **Real-time Analysis**: Utilize a variety of TensorFlow.js models for real-time insights.
- **Multiple Computer Vision Models**:
    - **MobileNet**: Classify images into 1,000 common object categories.
    - **COCO-SSD**: Detect and localize 80 common objects in images and video streams.
    - **BlazeFace**: A lightweight and performant face detector.
    - **Handpose**: Detect hands and predict 21 keypoints in real-time.
    - **PoseNet**: Estimate human poses by detecting key body joints.
- **Interactive UI**: A clean, dark-themed interface built with ShadCN UI and Tailwind CSS for an intuitive user experience.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (with React)
- **AI/ML**: [TensorFlow.js](https://www.tensorflow.org/js)
- **UI Components**: [ShadCN UI](https://ui.shadcn.com/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: TypeScript

## Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later recommended)
- `npm` or `yarn`

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/visionary.git
    cd visionary
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application running.
